# Информационная система супермаркета

Проект представляет собой мобильное приложение, интегрированное с внутренними информационными системами супермаркета. Основная цель — устранить ключевые проблемы, с которыми сталкиваются покупатели: неактуальные ценники на полках, неудобство использования физических карт лояльности, отсутствие гибких персональных скидок.

Решение позволяет пользователю в реальном времени получать достоверную информацию о ценах через сканирование штрихкода, управлять цифровой картой лояльности непосредственно из приложения и самостоятельно выбирать товары, на которые будут действовать персональные скидки.

## Стек технологий
- **Frontend:** React Native
- **Backend:** Django (Python)
- **База данных:** PostgreSQL
- **Веб-сервер:** Apache + mod_wsgi
